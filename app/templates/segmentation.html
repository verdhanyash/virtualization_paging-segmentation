ADDRESS TRANSLATION MECHANICS
Logical Address: A two-part tuple [segment number | offset] where the segment number indexes into the Segment Table and the offset is the byte position within that segment.
Hardware Check: The MMU extracts Base and Limit from the segment table entry. If offset < Limit, the Physical Address = Base + offset. Otherwise, a TRAP is raised.
On TRAP: The CPU raises an interrupt → the OS receives a segmentation fault signal → the offending process is typically terminated or a signal handler is invoked.
