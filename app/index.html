Key Concepts
Page Fault
When a referenced page is not in any frame, a page fault occurs. The OS must load it from disk — an expensive operation.
Bélady's Anomaly
A counter-intuitive phenomenon in FIFO where increasing the number of frames can increase page faults for certain reference strings.
External Fragmentation
Free memory is split into small, non-contiguous holes between allocated segments — enough total free space, but no single hole is large enough.
